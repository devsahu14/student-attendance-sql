# Student Attendance Management System (SQL)

## Overview
This project is a SQL-based Student Attendance Management System designed to manage student details, courses, and attendance records. It generates attendance reports and calculates attendance percentages.

## Features
- Store student and course information
- Record daily attendance
- Calculate attendance percentage
- Identify students with low attendance
